INTRODUCTION

These auxiliary files contain the source code, sample input files and associated
description of the KRC numerical model, all in plain text. There is a
full-resolution map of the computed annual average surface temperature on Mars
corresponding to Figure 10 in the article.

2012JE004164-ds12.bin5  is a 51.8 Mbyte binary file of Mars average annual
surface temperature; full resolution version corresponding to Figure 10.  The
map format is Simple-cylindrical projection, 1/20 degree resolution.  The file
contains a 512 byte ASCII header followed by 16-bit integer array, 7200 samples
by 3600 lines, of scaled annual average surface kinetic temperature.  The
scaling from integer to temperature in Kelvin is T=I*160.+200.  [original file:
TsaTiltP.bin5]
