## Common Differences Found

Based on analysis of the input file diffs, the most common differences are:

1. **Trailing whitespace** in PORB rotation matrix (cosmetic, no impact)
2. **Output file paths** (expected - PyKRC uses `./krc.t52`, Davinci uses full path)
3. **Parameter ordering/presence** in changecards (some tests show different changecard sets)
4. **Numerical precision** in some parameters (rare, typically rounding differences)

## Viewing Diffs

In VS Code or GitHub, the diff syntax highlighting will show:
- **Red (-)**: Lines only in PyKRC
- **Green (+)**: Lines only in Davinci
- **White**: Unchanged lines (context)
